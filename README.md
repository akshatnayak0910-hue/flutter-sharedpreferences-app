# Flutter SharedPreferences App

A simple and clean SharedPreferences App built using Flutter and Dart.

This project demonstrates the basic concepts of Flutter development such as:

* SharedPreferences package usage
* Local data storage
* Save, Get, and Remove operations
* TextEditingController
* StatefulWidget
* UI design using TextField, ElevatedButton, Column, and Scaffold

## Features

✔ Save data locally
✔ Retrieve saved data
✔ Remove stored data
✔ SharedPreferences integration
✔ Clean and modern UI
✔ Responsive design
✔ Beginner-friendly Flutter project

## Technologies Used

* Flutter
* Dart
* Material Design
* SharedPreferences Package

## Required Dependency

Add this package inside your `pubspec.yaml` file:

```yaml id="sp002"
dependencies:
  flutter:
    sdk: flutter
  shared_preferences: ^2.2.2
```

Then run:

```bash id="sp003"
flutter pub get
```

## Output Screen

The app contains:

* AppBar with title
* Input field for data entry
* Save button
* Get button
* Remove button
* Stored value display section
* Custom footer text
